## Proyectos en producción

### FagSol — Escuela Virtual

**Stack:** Django 5, Next.js 14, PostgreSQL, Docker, Redis, Celery, Mercado Pago, JWT · **Hosting:** Azure

Plataforma educativa en producción: cursos, roles (estudiante, instructor, admin), registro e inicio de sesión (JWT), pasarela de pagos (Mercado Pago, tarjetas reales), multi-moneda, certificados, dashboards y webhooks. Clean Architecture en backend, frontend por features.

### EduTravel del Perú

**Stack:** PHP, MySQL, HTML, CSS, Tailwind · **Hosting:** Hostinger

Sitio web para tour operador con +50 años de experiencia. Turismo nacional, internacional y estudiantil; programas por regiones y ciudades, cotización rápida, formulario de contacto, WhatsApp, panel de administración. Estructura MVC.

### SolInd — E-Commerce Industrial

**Stack:** HTML, CSS, Bootstrap 5, PHP, MySQL, Mercado Pago · **Hosting:** cPanel

Plataforma corporativa + e-commerce para empresa industrial peruana: catálogo, carrito, checkout, panel de administración (categorías, productos, pedidos), sesiones, CSRF, pasarela de pagos.
